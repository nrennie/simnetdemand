## Installation

Install using:
``` r
devtools::install_github("nrennie/simnetdemand")
```
or
``` r
remotes::install_github("nrennie/simnetdemand")
```

## How To Use

* Simulate and forecast demand
* Calculate booking limits
* Simulate booking process
